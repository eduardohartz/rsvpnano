# RSVP Nano v1.0.0

First stable release. This release restores touch playback controls, adds speaker feedback and
quality-of-life features, and lands a firmware-wide bug-fix sweep.

## Highlights

- Restored touch playback on the reading screen: double-tap toggles play/pause, press-and-hold
  starts playback while paused or pauses at the end of the sentence while playing.
- Restored swipe-right Back navigation throughout menus, including text entry.
- Reworked the scrub-preview browse gesture into direct drag-to-scroll with flick momentum, and
  capped preview re-renders to remove flicker during fast drags.
- Added chapter tick marks and a progress track under the scrub preview.
- Added UI sounds through the speaker for menu moves, selections, back navigation, play/pause,
  and the low-battery warning, with a Sounds toggle in Display settings.
- Added lifetime reading stats (words read and reading time) to the footer label cycle.
- The main menu Resume entry now shows the current book title and progress.
- Added a sleep timer (15/30/45/60 minutes) that pauses playback at a sentence end.
- Standby now uses the screen-off saver below 15% battery regardless of the configured
  screensaver.
- Bug-fix sweep across the firmware: touch recovery after controller glitches, touch handling
  during companion sync, index builds aborted by low memory no longer commit truncated books,
  USB mass-storage no longer hangs on out-of-range reads, OTA metadata downloads time out
  instead of hanging, RSS feeds survive bodyless items and entity-escaped HTML, EPUB conversion
  no longer swallows text around bare ampersands, and `.rsvp` metadata parsing handles long
  titles and missing trailing newlines.
- OTA updates now default to the `eduardohartz/rsvpnano` repository.
